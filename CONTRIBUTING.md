# Contributing

Bug reports and pull requests are welcome. Real-world cases — actual posting
text that misparsed — make the best issues and the best tests.

## Ground rules

- Keep the library dependency-free (Python standard library only).
- Every behavior change needs a test that fails without it.
- This library also runs inside [SearchSteward](https://searchsteward.com)'s
  production pipeline, so changes are reviewed with that bar.

## Developer Certificate of Origin

By submitting a pull request you certify the
[Developer Certificate of Origin](https://developercertificate.org/): that you
wrote the contribution or otherwise have the right to submit it under this
project's MIT license.

Sign off each commit with `git commit -s`, which adds a `Signed-off-by` line.
